# Ense_Pulsa

Android OCR app to scan pulsa/credit voucher numbers using CameraX and ML Kit.

## Features
- CameraX live preview with draggable selection overlay
- ML Kit text recognition for automatic digit extraction
- ISP selection (Telkomcel, Telemor, TT)
- One-tap dial with USSD code

## Build
Open in Android Studio, sync Gradle, and run.
